Propósito

El propósito de este sitio Web en tener claridad de los conceptos dey el uso de Flex Box y Grid CSS en el uso como herramienta en HTML5

Flex Box

Flex box: Es una herramienta para maquetar o dar una organización visual de la pagina web esto parte de que es flexible como su nombre lo indica orientado en una dimensión da solvencia a varios problemas de maquetar con un buen soporte y con el uso de grid hace mas funcional la herramienta dando un mayor rango de usos. Para el uso de Flex hay que separar y identificar el contenedor de sus hijos. Contenedor es un elemento al que denominamos display:flex haciendo de que todos los hijos de este elemento estén en modo flex y de esta manera reciben propiedades flex.

CSS Grid

Display: grid; CSS Grid es un mecanismo flexible de maquetar en dos dimensiones independiente de su origen. Al igual que flex se ajusta en los pixeles en pantalla, solo que lo hace de una manera mas inteligente por asi decirlo ya que calcula las unidades y los porcentajes asignados según el creador. y en cuando a sus hijos se pueden alinear y justificar de una manera mas dedicada.
