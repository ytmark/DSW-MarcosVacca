Flexbox y Grid CSS

Son herramientas o metodos para dar forma y orden a los elementos de una pagina WEB. Cada uno con sus funciones especificas Flex Box en una sola dimensión CSS Grid en dos dimensiones.

En el menu/sección temas encontraremos claridad y uso de cada uno de los métodos.

Propósito

El propósito de este sitio Web es tener claridad de los conceptos y el uso de Flex Box y Grid CSS en el uso como herramientas en HTML5.
